# Attributions

The omnigen-vault **runtime** (CLI, MCP server, HTTP gallery server, SQLite vault) has
**zero npm dependencies** — it uses only Node.js built-in modules.

The **web gallery** (`web/`) is built with the following open-source projects:

| Project | Purpose | License |
|---|---|---|
| [Vite](https://vitejs.dev/) | Build tool / dev server | MIT |
| [React](https://react.dev/) | UI library | MIT |
| [Tailwind CSS](https://tailwindcss.com/) | Styling | MIT |
| [shadcn/ui](https://ui.shadcn.com/) | Component patterns | MIT |
| [Radix UI](https://www.radix-ui.com/) | Accessible UI primitives | MIT |
| [lucide](https://lucide.dev/) | Icons | ISC |
| [TanStack Query](https://tanstack.com/query) | Data fetching/caching | MIT |
| [cmdk](https://cmdk.paco.me/) | Command menu | MIT |
| [Sonner](https://sonner.emilkowal.ski/) | Toasts | MIT |

The macOS menu-bar app uses Apple **SF Symbols** (free for use in apps).

App icon and showcase images under `docs/assets/` are AI-generated.
